# Source Data Workbooks

The journal submission included four source-data workbooks:

```text
2136_3_source_data_24008_tgxpg5.xlsx  # Figure 2
2136_3_source_data_24009_tgxpg5.xlsx  # Figure 3
2136_3_source_data_24010_tgxpg5.xlsx  # Figure 4
2136_3_source_data_24011_tgxpg5.xlsx  # Figure 5
```

The complete workbooks are not included in this public code repository.

Rationale:

- Figure 4 and Figure 5 source workbooks contain human-subject and participant-level measurements.
- The manuscript data availability statement describes the study data as available from the corresponding authors upon reasonable request.
- This repository is intended to provide minimal reproducible code, not to redistribute restricted source data.

The only source-data-derived file included here is:

```text
data/figure5hi_reported_summary.csv
```

It contains only aggregate model-level Figure 5h-i values and does not include participant-level rows.
